# DummyOutput

## Description

The DummyOutput module lets you play with the system without purchasing hardware.
It's good for practicing with the project and testing plugins

## Example configuration

```
output:
  - module: DummyOutput
    relays:
      0: DummyOutput0
      1: DummyOutput1
      2: DummyOutput2
      3: DummyOutput3
```
